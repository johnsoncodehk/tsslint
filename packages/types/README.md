# @tsslint/types

Shared TypeScript types for TSSLint rules, plugins, and config — `Config`, `Rule`, `Plugin`, `Reporter`, `LinterContext`, and friends.

Usually pulled in transitively via [`@tsslint/config`](../config). Install directly only when authoring a plugin without going through `@tsslint/config`:

```bash
npm install @tsslint/types --save-dev
```

See the [root README](../../README.md) for rule and plugin authoring.
